# Behavior Data Dashboard

## 📌 Overview
This project analyzes behavior and attendance trends across campuses to identify disproportionality, root causes, and performance patterns. The dashboard provides drill‑downs for campus, grade, and incident types.

## 🛠 Tools & Technologies
- Power BI  
- Power Query  
- DAX  
- Excel  

## 📊 Key Features
- Data cleaning and validation  
- Data modeling and relationships  
- DAX measures for KPIs  
- Interactive visuals and drill‑downs  
- Insight storytelling for leadership  

## 📁 Project Structure
- `data/` → sample or dummy datasets  
- `pbix/` → Power BI file (optional)  
- `images/` → dashboard screenshots

1. Open the Power BI file (when added)  
2. Load sample data from the `data/` folder  
3. Refresh queries
